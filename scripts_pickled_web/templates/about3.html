Machine learning for detecting sarcastic Tweets


Sarcasm is an effective communication tool, with speakers usually communicating sarcasm using body language or by altering their tone of voice. In written communication, sarcasm can be especially difficult to detect because written communication lacks a systematic method for writers to communicate their sarcastic intent. Although it is an important source of data for sentiment analyses, social media often contains sarcastic posts which if not correctly interpreted, represent a potentially large source of error.

I addressed the sarcasm problem using natural language processing and machine learning to construct and tune a model for predicting sarcastic tweets. Users can test the model for themselves using the web app. I encourage those interested to read more about the model and it's practical business applications related to sentiment analysis in the report.
